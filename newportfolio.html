Front-end Projects
The Simpsons Clicky Game

The Simpsons Clicky Game is a React application that tests your memory with a Simpsons-themed memory game. The user tries to click on all twelve cards without clicking the same card twice. The cards are shuffled each time the user clicks on a card. How good is your memory?

Bar Finder

The first project I worked on with a team. This web application utilizes the Google Maps API and two other APIs to allow a user to search for a bar based on location and bar type. Users may also search for drink recipes so that they may make the drinks themselves. By creating an account and logging in, users can select bars and drinks and add them to their Favorites.

 Deployed  GitHub Repo
Train Scheduler

An application that provides up-to-date information about various trains, namely their arrival times and how many minutes remain until they arrive at their station. The time of the train's next arrival is calculated in relation to the current time, which is powered using Moment.js. Firebase is used to host arrival and departure data.

 Deployed  GitHub Repo
GifTastic

This dynamic web page populates with gifs using the GIPHY API. As the user searches for a topic, the page populates with buttons by topic. When each topic button is clicked, ten gifs are displayed. The user can change the state of each gif from "still" to "animate", and vice versa, by clicking on the gif.

 Deployed  GitHub Repo
Crystal Collector

This game combines HTML and CSS techniques with JavaScript and jQuery. Try to match the random number by clicking on the four different crystals, each with its own random value set at the start of play. Flash messages display when the user wins or loses.
